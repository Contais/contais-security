# contais-security

## 简介

Contais-security是一个能100%满足企业主流认证授权所需的功能的工具包。
```
- 支持两种前端类型
    浏览器（Session方式）
    APP（Token方式） 

- 支持多种授权方式
    配置静态权限
    依赖数据库服务提供动态权限 
    
- 支持多种身份认证方式
    用户名密码登录
    第三方(微信、QQ等)登录
    手机短信登录等 
    
- 支持分布式系统
    集群Session管理
    SSO单点登录
```
-------------------------------------------------------------------------------

## 包含组件
- contais-security-core         核心包    
- contais-security-app          app端
- contais-security-browser      浏览器端
